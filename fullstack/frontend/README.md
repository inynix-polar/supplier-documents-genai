# Frontend — Regex Lab (React + TypeScript)

Здесь пусто намеренно — scaffolding фронта часть задачи.

```bash
npm create vite@latest . -- --template react-ts
npm install
npm run dev
```

Бэкенд: `../backend` (эндпоинт `POST /api/regex/suggest`, CORS открыт,
работает с `LLM_FAKE=1` без ключа).

Минимум, который должен быть на странице:
- textarea с исходным текстом;
- список позитивных примеров (что матчить) и анти-примеров (что не матчить);
- кнопка «Сгенерировать regex» → `/api/regex/suggest`;
- редактируемое поле паттерна + флаги;
- **живая подсветка** совпадений в тексте (зелёным) и анти-примеров, попавших под
  паттерн (красным) — пересчёт на каждый ввод/правку;
- индикатор «паттерн покрывает все позитивы и не задевает негативы»;
- кнопка «Скопировать YAML-блок» (формат см. `../backend/configs/attribute_example.yaml`).
